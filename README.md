Tocador de Acordes: Swing 130
O Tocador de Acordes: Swing 130 é uma ferramenta web interativa desenvolvida para auxiliar estudantes e compositores na criação, visualização e exportação de estruturas harmônicas.
Desenvolvido para a disciplina Criação Musical 1, do Departamento de Artes e Comunicação (DAC) da UFSCar, o projeto tem como foco a sonoridade swing e a análise harmônica funcional.
________________________________________
🚀 Funcionalidades
🎹 Harmonia e Execução
Grade de Acordes
Seleção intuitiva de tétrades (maj7, 9, 7b9, m7, m7b5, dim7) em todas as tonalidades cromáticas.
Motor de Áudio
Execução de amostras de áudio com articulação swing.
Controle de Reverb
Ajuste dinâmico da ambiência sonora.
Sequenciador
Organização de acordes em compassos (8 por linha), com funções de substituição e remoção.
________________________________________
🔍 Análise e Auxílio Visual
Campos Harmônicos
Destaque visual automático para o campo harmônico diatônico maior e menor de Dó.
Cifragem por Graus
Alternância instantânea entre cifras literais (ex.: Cmaj7) e numerais romanos (ex.: Imaj7), baseados em Dó.
Modos de Tema
Suporte para Tema Escuro (padrão) e Tema Claro, com estética vintage suave.
________________________________________
📤 Exportação e Interoperabilidade
O projeto oferece suporte a diversos formatos para integração com outros softwares musicais:
Projeto (.txt)
Salva e abre o estado completo do trabalho (título, autor e acordes).
Imagem (PNG)
Gera uma partitura harmônica visual pronta para compartilhamento ou impressão.
XML (MusicXML)
Exportação em compasso 4/4, compatível com MuseScore, Finale e Sibelius.
Tabela
Formato de texto organizado para documentação.
BIAB
Formato compatível com o software Band-in-a-Box.
________________________________________
🛠️ Tecnologias Utilizadas
•	HTML5 e CSS3 — layout responsivo com Tailwind CSS
•	JavaScript moderno — manipulação do DOM e lógica harmônica
•	Web Audio API — processamento de áudio em tempo real e reverberação convolutiva
•	Canvas API — geração dinâmica de imagens para exportação
